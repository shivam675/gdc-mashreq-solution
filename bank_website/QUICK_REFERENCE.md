## Common Questions

**Q: Why Ollama instead of OpenAI?**
A: Local inference, no API costs, privacy, works offline

**Q: Why WebSocket?**
A: Real-time updates, streaming LLM responses, live operator awareness

**Q: Why human approval?**
A: Critical PR content needs human judgment, AI augmentation not replacement

**Q: Can you add more agents?**
A: Yes! Architecture supports unlimited agents, just add to workflow

**Q: Production ready?**
A: Demo-ready. For production: PostgreSQL, auth, monitoring, tests, deployment
